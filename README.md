# POS Papelería

![Banner](banner.png)

Un sistema de Punto de Venta (POS) moderno y eficiente, diseñado específicamente para papelerías. Construido con una arquitectura robusta (Node.js, Express, Prisma, SQLite) en el backend y una interfaz PWA dinámica (React, Vite, Tailwind CSS) en el frontend.

## 🚀 Características Principales

- **POS Inteligente**: Búsqueda en tiempo real, carrito con cálculo de IVA (16%) y procesamiento rápido de cobros.
- **Gestión de Inventario**: CRUD completo de productos y edición de stock de manera rápida.
- **Historial de Ventas**: Registro detallado de ventas, estadísticas y paginación para un análisis fácil.
- **Diseño Premium**: Interfaz moderna, minimalista y oscura (Dark Theme) enfocada en la experiencia del usuario.

## 🛠 Guía de Instalación Rápida

### Requisitos Previos
- **Node.js** v18 o superior
- **npm** v9 o superior

### 1. Backend (API)
```bash
cd backend
npm install
npx prisma generate
npx prisma db push
npm run seed
npm run dev
```
El servidor iniciará en **http://localhost:3001**

### 2. Frontend (Interfaz)
En una nueva terminal:
```bash
cd frontend
npm install
npm run dev
```
La aplicación iniciará en **http://localhost:5173**

Para más detalles, consulta la [Guía de Instalación Completa](GUIA_INSTALACION.md).
